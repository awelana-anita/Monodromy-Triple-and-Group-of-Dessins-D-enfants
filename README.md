# Monodromy-Triple-and-Group-of-Dessins-D-enfants
An algorithm that generates the monodromy triple (sigma_0, sigma_1, sigma_infinity) and group from a given dessin using SageMath. The algorithm processes the graph and constructs sigma_0 and sigma_1 by traversing the edge labels counterclockwise around the grey and red vertices respectively, and sigma_infinity by traversing clockwise around faces.
