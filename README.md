# Monodromy-Triple-and-Group-of-Dessins-D-enfants
A group is a set satisfying four properties. A graph is a visualization of how things are related. Dessin d’enfants are connected, loopless, bipartite graphs embedded on a compact Riemann surface. The goal of this project is to generate an algorithm that constructs the monodromy triple (sigma_0, sigma_1, sigma_infinity) and group from a given dessin using SageMath. The algorithm processes a graph, labels the edges, colors the vertices through partition and embeds the graph. It then constructs sigma_0 and sigma_1 by traversing the edge labels counterclockwise around the grey and red vertices respectively, and sigma_infinity by traversing clockwise around faces.
